# Student Information Management Application
This project is a simple application to manage student information that I did when I learned Firebase Firestore.

### Features
- Sign up
- Sign in
- Change profile picture
- Manage user: CRUD
- View login history
- Manage student: CRUD
- Sort student list based on various criteria
- Search students by multiple criteria
- View student details
- Manage certificates of a student: CRUD
- Export the list of students to csv file
- Export the list of certificates to csv file
